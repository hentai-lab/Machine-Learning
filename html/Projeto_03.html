Esta uma das clulas mais importantes pois onde executo um lao de repetio responsvel pela criao de vrios modelos com o intuito de achar o melhor. Conforme os modelos so criados, seus resultados de r2 e a quantidade de rvores correspondente a esses resultados so armazenados em um dataframe o qual ser usado posteriormente para saber qual o melhor modelo.

In [0]:
results = pandas.DataFrame(columns=['rvores', 'r2'])
arvores = []
e_max = 1000
e_range = 20
i = 0

while(i < e_max):
  i += e_range
  arvores.append(i)

for arvris in arvores:
  rf = RandomForestRegressor(n_estimators=arvris, oob_score=True, max_features=2, random_state=400)
  rf.fit(X_train, y_train)
  results = results.append({'rvores': arvris, 'r2': rf.oob_score_}, ignore_index=True)

Nesta clula plotado um grfico de linha mostrando como se comportou cada modelo baseado em seu r2. de fcil visualizao o que est acontecendo atravs deste grfico! Seu eixo X o r2 e o eixo Y a quantidade de rvores!

Esta clula responsvel apenas para identificar qual o maior r2 e, consequentemente, achar o ndice da linha dentro do dataframe correspondente a este r2 para poder achar a quantidade de rvores do melhor modelo com o intuito de utiliz-lo para predio.

rr2 = results['r2'].max()
rtree = results.loc[results.loc[results.r2 == rr2].index, ['rvores']].values[0][0]
